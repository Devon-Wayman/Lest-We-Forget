How to use this asset
---------------------

A LUT is a special texture used to transform the colors of an image giving a special look.
This asset contains +200 LUTs including artistic/cinematic LUTs and tints.

To use a LUT, load it into Beautify's LUT section.
Also you can adjust the LUT intensity and other color grading options of Beautify to obtain the perfect look based on your visual preferences.

Note: the LUT format used in this asset is 1024x32 pixels with dark/zero values at the bottom. This format is 100% compatible with Beautify asset:
https://assetstore.unity.com/packages/vfx/shaders/fullscreen-camera-effects/beautify-2-163949?aid=1101lGsd
